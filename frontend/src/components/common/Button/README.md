# 버튼 컴포넌트 사용법
```jsx
<Button variant={Button.Variants.PRIMARY} size={Button.Sizes.LARGE}>
  기본 버튼
</Button>

<Button variant={Button.Variants.OUTLINE} loading={true}>
  로딩 중인 아웃라인 버튼
</Button>

<Button 
  variant={Button.Variants.DANGER} 
  size={Button.Sizes.SMALL}
  disabled={true}
>
  비활성화된 위험 버튼
</Button>
```
